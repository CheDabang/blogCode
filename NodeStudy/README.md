## 基础知识篇

### Node初步认识

+ Node 是什么？
    - Node 是一个让 JavaScript 运行在服务端的开发平台，它让 JavaScript 成为与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语言。

+ Node中的JavaScript
    - 没有BOM、DOM
    - ECMAScript 基本的 JavaScript语言部分
    - 打破次元壁垒，可以在服务器商快乐奔跑了。
    - 浏览器是无法识别在Node上面运行的JavaScript代码
